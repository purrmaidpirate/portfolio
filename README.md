# Ranjani Ramakrishnan — Portfolio

Personal portfolio site built with React, Three.js, and Vite. Features an infinite 3D canvas for browsing work, plus pages for projects, blog, and about.

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React 19
- Three.js / React Three Fiber
- TypeScript
- Vite
